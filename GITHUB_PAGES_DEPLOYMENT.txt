═══════════════════════════════════════════════════════════════════════
METHOD 1: USING GITHUB WEB INTERFACE (EASIEST!)
═══════════════════════════════════════════════════════════════════════

STEP 1: Go to Your Repository
Visit: https://github.com/RaghvendraDehariya/raghav-hair-salon

STEP 2: Enable GitHub Pages
1. Click "Settings" tab (top right)
2. Click "Pages" (left sidebar)
3. Under "Build and deployment":
   - Source: Select "GitHub Actions"
4. Click "Create your own" or "Configure"

STEP 3: Create GitHub Actions Workflow
GitHub will open a file editor. Paste this:

```yaml
name: Deploy to GitHub Pages

on:
  push:
    branches: [ main ]

jobs:
  build-and-deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      
      - name: Setup Node
        uses: actions/setup-node@v3
        with:
          node-version: '18'
          
      - name: Install and Build
        run: |
          cd frontend
          yarn install
          yarn build
          
      - name: Deploy
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./frontend/build
```

4. Click "Commit changes"
5. Wait 2-3 minutes

✅ Your website will be live at:
https://RaghvendraDehariya.github.io/raghav-hair-salon/

═══════════════════════════════════════════════════════════════════════
METHOD 2: MANUAL UPLOAD (IF METHOD 1 DOESN'T WORK)
═══════════════════════════════════════════════════════════════════════

STEP 1: Create gh-pages branch
In your repository:
1. Click "main" dropdown (top left)
2. Type: gh-pages
3. Click "Create branch: gh-pages"

STEP 2: Upload Build Files
1. Switch to "gh-pages" branch
2. Click "Add file" → "Upload files"
3. Download the build folder from Emergent
4. Upload ALL files from /app/frontend/build/
5. Commit changes

STEP 3: Enable Pages
1. Go to Settings → Pages
2. Source: "Deploy from branch"
3. Branch: "gh-pages"
4. Folder: "/ (root)"
5. Save

✅ Website live in 5 minutes!
